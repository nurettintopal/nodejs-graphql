==============================================
GraphQL Server Boilerplate for Node.js

![](https://imgur.com/lIi4YrZ.png)


## usage

install dependencies:
```sh
$ npm install
```

run:
```sh
$ npm run start
```

open a browser and go to:
```sh
http://localhost:4000/graphql
```

## tests

run:
```sh
$ npm run test
```

## build

run:
```sh
$ npm run build
```
