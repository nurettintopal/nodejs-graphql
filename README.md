Node.js GraphQL Server
==============================================
GraphQL Server Boilerplate for Node.js

![](https://imgur.com/lIi4YrZ.png)


## usage

install dependencies:
```sh
$ npm install
```

run:
```sh
$ npm run dev
```

open a browser and go to:
```sh
http://localhost:2020/graphql
```
